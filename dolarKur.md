#<h1 align="center">Python Dolar Bot<h1>

##Requirements

* request module
* BeautifulSoup
* sqlite3
* notifypy
* time
* datetime

#installation
for request module
```
pip install requests
```
for BeautifulSoup module
```
pip install beautifulsoup4
```
for notifypy module
```
pip install notify-py
```
